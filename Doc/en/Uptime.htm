Uptime

Description

Uptime agent is used to monitor the uptime of a server.

Configuration

  • setting directive syntax : 

    <setting name="" value="" />

    name value
    alarmcolor You can set the alarm color you want to use if their is an alert. It should be set to red or yellow. Default is yellow.
    delay You can set the delay used to compare with the last startup of the computer. Unit is seconds. You can specify a unit after the numeric value : "m" for minutes, "h" for hours, "d" for days. Default is 30 minutes
    testname You can set the testname that will be sent to the monitoring server (column name in the bbdisplay) Default is "uptime"

Examples

  • Uptime will warn during one hour with a red color : 

    <uptime>
	<setting name="delay" value="1h" alarmcolor="red" />
	<setting name="alarmcolor" value="red" />
</uptime>

Preparing Your BBWin Development Machine

BBWin Development Machine

Required software

  • Microsoft Visual C++ 2010 Express
  • Boost C++ libraries
  • Windows Subversion client

Recommended software

  • Windows Installer XML (WiX) toolset (Version 3.0 or later)
  • Microsoft HTML Help Workshop
  • Microsoft Visual Web Developer 2010 Express
  • Text Editor with syntax highlighting

Software Installation

  1. Install Visual C++ Express and apply the Visual Studio Service Pack 1. Note: if you are installing other Visual Studio 2010 Express components (such as Visual Web Developer), install those first before applying the service pack.

    Download Visual Studio Express for free from the Microsoft website and Service Pack 1 from their Download Center.
  2. Install the Boost C++ libraries and add BoostInstallDir to the user or system environment variables with the value set to the Boost installation folder (e.g. BoostInstallDir=C:\Boost).
  3. Install a Windows Subversion client. TortoiseSVN is highly recommended for this - it integrates into Windows Explorer, adding entries to the context menu for files and folders. TortoiseSVN can be downloaded from here.
  4. If you want to package the compiled executables into an MSI, install WiX and the Microsoft HTML Help Workshop. The latter creates the compiled help file BBWin.chm, which is then included in the MSI.
  5. For additions and modifications to the HTML help files, Visual Web Developer is an excellent tool - providing syntax highlighting, code folding, auto-completion and validation. See step 1 for download instructions.
    An additional light-weight editor (such as Notepad++ or Crimson Editor) is useful for editing BBWin.cfg and other files. These provide syntax highlighting for the XML, making it much easier to read. Visual Web Developer can be used for this, but does take quite a bit longer to load so can be tedious for small modifications.

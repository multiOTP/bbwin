Support

BBWin Project

You can follow the BBWin project via the sourceforge project page where you will be able to download the lastest release. BBWin page.

Technical support for BBWin is available on the BBWin forum and the BBWin mailing list.

Also, because BBWin is closely tied to the Xymon monitoring project, the next paragraph gives you the important links to this project

For troubleshooting, you may post in the help forum. To facilitate the diagnostic, please post your request with these elements :

  • The Windows version you are using (Windows Version, Service Pack Number, Language)
  • BBWin installation status : BBWin version and the BBWin.cfg file

Xymon Project

The lastest Xymon release can be download from its sourceforge page. Official Xymon project webpage is here. Online demonstration is available here.

Technical support for Xymon is available on its official mailing list xymon@xymon.com.

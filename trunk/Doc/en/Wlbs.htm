Wlbs

Description

Wlbs vbscript is used to monitor WLBS (NLB) clusters. It will warn depending :

  • the state

Usage

Wlbs vbscript uses a configuration file called wlbs.cfg which is located in BBWin etc directory at installation. It define the default node state of the server monitored.

Configuration Format

Rule syntax


DEFAULTNODESTATUS <StatusType>

DEFAULTNODESTATUS will set the default status of the node. WLBS.vbs will signal alerts by comparing the current status with the default status. If not defined, the default status is set to started.

Possible StatusType are :

  • stopped: node stopped
  • started: node started
  • suspended: node suspended
  • converged: node converged

Examples

  • Here an example of report : 
    
wlbs status

IP : 77.78.79.22

Name : 77.78.79.30:5

Prority : 5


    Default Status : converged


    Current Status : stopped

Notes

  • This external is not installed by default
  • You may add comments in the wlbs.cfg configuration file, by using the character '#' before your comments
